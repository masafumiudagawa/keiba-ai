2. 追加すべきデータカテゴリ（12項目）

A. 全馬の完全な戦績 最優先

現在3-10走/馬 → 全キャリア（新馬〜直近）を網羅すべき。特に条件戦での成績が欠落。

不足: ミュージアムマイル(4走のみ), シュガークン(3走), マイネルエンペラー(3走) → netkeiba/JBIS から全走を再取得

B. 上がり3Fタイム 最優先

現在ほぼ全レースで欠損。宝塚記念は「持続力+末脚」が問われるレースで、上がり3Fは最重要ファクターの一つ。

必要: 各馬の直近5走の上がり3F 例: クロワ 33.5, 34.2, 33.8, ... → 末脚の安定性・爆発力を定量化

C. 通過順位（ポジション） 重要

各コーナーでの位置取り。脚質（逃げ/先行/差し/追込）の客観データ。

形式: "3-3-2-1" (1角-2角-3角-4角) → 脚質コードの精度向上 → ペース適性の分析

D. 馬体重推移 重要

馬体重と増減は体調のバロメーター。大幅増減は不調サイン。

必要: 直近5走の馬体重と増減 例: 514(+8), 506(-2), 508(+4), ... → ベスト体重からの乖離を計算

E. 血統詳細（種牡馬の距離・馬場適性） 重要

現在は父名のみ。種牡馬の産駒成績（芝2200m勝率、阪神勝率、重馬場勝率）が有用。

追加データ: キタサンブラック産駒: 芝2000-2400m勝率18%, 阪神15%, 重馬場12% エピファネイア産駒: 芝2000-2400m勝率16%, 阪神14%, 重馬場10% ゴールドシップ産駒: 芝2200+m勝率20%, 阪神18%, 重馬場22%(!) スワーヴリチャード産駒: 芝2200m勝率14%, 阪神12% → ゴールドシップ産駒は重馬場に強い = メイショウタバルの強み

F. 対戦成績（Head-to-Head） 重要

同じレースで走った馬同士の直接対決結果。

例: クロワ vs ダノンデサイル 大阪杯: クロワ1着, ダノン3着 → クロワ勝ち JC: クロワ4着, ダノン3着 → ダノン勝ち → 対戦成績 1勝1敗（互角） 例: レガレイラ vs メイショウタバル 宝塚記念: メイショウ1着, レガレイラ11着 → メイショウ圧勝 → このコース・距離での相性

G. スピード指数 / レーティング 有用

各レースの走破タイムを馬場差・ペース差で補正した「真の能力値」。

計算: 基準タイム - 走破タイム + 馬場補正 + ペース補正 → JRA-VAN / netkeiba のスピード指数を参照 → 異なる条件のレースを横比較可能

H. 調教師・厩舎の成績 有用

調教師の勝率、G1実績、阪神成績。

例: 友道厩舎 (ダノンデサイル): G1勝率高い 石橋厩舎 (メイショウタバル): 宝塚記念で実績 → 厩舎力を5段階で評価

I. 枠番バイアス（木曜以降） 有用

阪神芝2200mの枠順別成績。内枠/外枠の有利不利。

過去データ(阪神芝2200m): 1-2枠: 勝率8.5%, 複勝率22% 3-4枠: 勝率10.2%, 複勝率26% 5-6枠: 勝率9.8%, 複勝率24% 7-8枠: 勝率7.5%, 複勝率20% → 中枠がやや有利

J. パドック・馬体評価（当日） 当日

当日のパドックでの馬体チェック結果。毛艶・歩様・テンション。

ソース: YouTube生中継、競馬新聞のパドック評価 → 当日朝〜発走前に最終更新 → A/B/C/Dの4段階評価

K. 展開予測（ペースメーカー分析） 有用

逃げ馬の存在と想定ペース。ハイペース/スローペースで有利な馬が変わる。

今回の逃げ馬候補: メイショウタバル（昨年逃げ切り） シェイクユアハート（逃げ脚質） → 2頭いるのでミドル〜ハイペース想定 → 差し馬（レガレイラ、ダノンデサイル）に有利か

L. 休養明け分析 有用

前走からの間隔と、その馬の休み明け成績。

例: ミュージアムマイル 前走: 有馬記念(12/28) → 宝塚(6/14) = 約6ヶ月 過去の休み明け: 皐月賞→天皇賞秋(6ヶ月空き) = 2着 ← 休み明けOK → 長期休養でも走る馬かどうか
