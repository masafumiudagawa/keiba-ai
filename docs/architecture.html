1. プロジェクト概要

01

予測結果一覧

53特徴量 + 7データソースから各馬の勝率・複勝率を算出し、表形式で一覧表示。リアルタイムでデータ更新に追従。

02

レースシミュレーション

モンテカルロシミュレーションでレース展開を生成し、馬番号が動くアニメーションで可視化。各馬の脚質・ペース・位置取りを再現。

03

買い目オプティマイザ

投資金額・許容リスクを入力すると、ケリー基準と期待値計算に基づいて最適な買い目（単勝/馬連/三連複等）を自動提案。

シミュレーションロジック（モンテカルロ法）

各馬のレース中のポジションを200mごとにシミュレーション
  • 各馬にスピード基礎値・脚質パラメータ（逃げ/先行/差し/追込）を設定
  • ペース（ハイ/ミドル/スロー）を確率的に生成
  • コーナーでの加速減速・内外差・位置取りロスを物理モデルで反映
  • ラスト600m（上がり3F）で各馬の末脚能力を適用
  • ランダム要素（馬場状態・展開運・不利）を付加して1000回試行
  • 1000回の中央値パターンを「代表シミュレーション」として動画化

アニメーション仕様

項目仕様
描画HTML5 Canvas (2D) + requestAnimationFrame (60fps)
コース阪神芝2200m 右回り楕円を上空ビューで描画
馬の表現番号付き円（馬番カラー）、先頭からの差を実スケール反映
区間200m刻み（合計11区間）で位置を補間描画
操作再生/一時停止/速度変更(1x/2x/4x)/リプレイ
情報表示右側にリアルタイム順位表 + 残り距離 + ラップタイム
レース後着順確定 → 写真判定風のフィニッシュ演出

6. 機能③ 買い目オプティマイザ

ユーザー入力

入力項目型説明
投資金額数値総投資額（例: 10,000円）
リスク許容度スライダー低（堅実）〜 高（大穴狙い）
馬券種別チェックボックス単勝/複勝/馬連/ワイド/三連複/三連単
オッズ自動取得 or 手動各馬券のオッズ
除外馬選択買いたくない馬を除外

最適化ロジック

ケリー基準 + 期待値ベースの買い目最適化
  1. 期待値計算: 各馬券の的中確率(AI予測) × オッズ = 期待値
  2. フィルタリング: 期待値 > 1.0 の馬券のみ抽出（プラス期待値）
  3. ケリー基準: 各馬券の最適投資比率を算出
    f* = (p × b - q) / b
    p=的中確率, q=1-p, b=オッズ-1
  4. リスク調整: ユーザーのリスク許容度でケリー比率を縮小（1/4〜1/1 Kelly）
  5. 金額配分: 投資金額を各馬券に配分（100円単位に丸め）
