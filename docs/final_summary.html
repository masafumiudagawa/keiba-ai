6. 買い目最適化ロジック

対応馬券（全7種）

単勝 / 複勝 / 馬単 / 馬連 / ワイド / 三連複 / 三連単

5つのモード

モード内容
3パターン比較堅実/バランス/大穴の3パターンを横並び比較
AI自動最適化リスク×馬券種別でケリー基準ベースの最適配分
軸馬流し1-2頭を軸に指定→相手をAIが選出
フォーメーション1着/2着/3着候補を個別選択→点数+回収率計算
ボックスN頭BOX→全組合せ+回収率計算

的中確率の計算式

馬連: P(A1着)×P(B2着|A1着) + P(B1着)×P(A2着|B1着) ワイド: P(A複勝)×P(B複勝|A複勝) × 補正 三連複: P(A複)×P(B複)×P(C複) × 組合せ補正 馬単: P(A1着)×P(B2着|A1着) ← 着順固定 三連単: P(A1着)×P(B2着|A)×P(C3着|A,B)

各買い目に表示する情報

金額 / オッズ / 的中率 / 的中時払い戻し額 / 期待値 + 収支シミュレーション

7. シミュレーションエンジン

方式: モンテカルロ法 500回試行 パラメータ: - base_speed: 0.993〜1.012（G1レベルの僅差） - 脚質(逃げ/先行/差し/追込)×区間(序盤/中盤/終盤)の速度倍率 - 当日の調子: normal(0, 0.005) のランダム - 馬群の不利: 中盤区間でランダム発生 - ノイズ: normal(0, 0.25) / 区間 結果: 先頭との相対位置 → Canvas 2Dで60fps描画
